help 서명 적용 방법
1

서명 복사하기

위의 '서명 복사' 버튼을 클릭하여 생성된 서명을 클립보드에 복사합니다.

2

Gmail 설정 열기

우측 상단의 톱니바퀴 아이콘을 클릭하고 '모든 설정 보기'를 선택합니다.

3

서명 붙여넣기 및 저장

'기본설정' 탭 → '서명' 섹션 → 복사한 서명을 붙여넣고 저장합니다.

1

서명 복사하기

위의 '서명 복사' 버튼을 클릭합니다.

2

Outlook 설정 열기

설정 (⚙️) → 'Outlook의 모든 설정 보기' → '메일' → '작성 및 회신'

3

서명 붙여넣기 및 저장

'+ 새 서명' → 이름 입력 → 편집 영역에 붙여넣기 → 기본 서명 지정 후 저장

1

HTML 코드 복사

위의 'HTML 코드' 버튼을 클릭하여 HTML을 복사합니다.

2

Mail 설정 열기

Apple Mail → 설정 (⌘,) → '서명' 탭 → '+' 클릭

3

서명 붙여넣기

서명 편집 영역에 ⌘+V로 붙여넣기합니다.
