# 📧 Ambience 이메일 서명 설정 가이드

안녕하세요! 아래 가이드를 따라 Ambience 이메일 서명을 5분 안에 설정할 수 있습니다.

---

## Step 1. 서명 생성하기

**서명 생성 페이지 접속 👉 [여기를 클릭](https://downy-ambience.github.io/ambience-signature/)**

아래 정보를 입력해주세요:

| 항목 | 설명 | 예시 |
|------|------|------|
| 이름 (Name) | 영문 이름 | `Downy Jung` |
| 직책 (Title) | 본인 직책 | `Producer` |
| 부서 (Department) | 드롭다운에서 선택 (선택 안함 가능) | `Production Dept.` |
| 국가코드 | 드롭다운에서 선택 | `🇰🇷 +82` |
| 전화번호 (Phone) | 번호만 입력 (자동 포맷됨) | `01033971686` |
| 이메일 (Email) | 회사 이메일 | `downy@ambienceseoul.com` |

> 💡 전화번호는 `01033971686` 처럼 그냥 쭉 입력하면 `+82 10-3397-1686` 형태로 자동 변환됩니다!

입력이 완료되면 오른쪽 미리보기에서 서명을 확인하세요.

---

## Step 2. 서명 복사하기

미리보기 상단의 **"📋 서명 복사"** 버튼을 클릭합니다.

> ✅ "서명이 복사되었습니다" 메시지가 표시되면 성공!

---

## Step 3. Gmail에 서명 적용하기

### 3-1. Gmail 설정 페이지로 이동

1. **Gmail** 접속 → 오른쪽 상단 ⚙️ **설정** 클릭
2. **"모든 설정 보기"** 클릭
3. **"일반"** 탭에서 아래로 스크롤 → **"서명"** 섹션 찾기

> 🔗 또는 이 링크로 바로 이동: **[Gmail 설정 > 일반](https://mail.google.com/mail/u/0/#settings/general)**

### 3-2. 새 서명 만들기

1. **"+ 새로 만들기"** 클릭
2. 서명 이름 입력 (예: `Ambience`) → **만들기**
3. 서명 편집 영역에 **Ctrl+V** (Mac: **⌘+V**) 로 붙여넣기

### 3-3. 기본 서명으로 설정

서명 섹션 하단에서:
- **"새 이메일에 사용할 서명"** → 방금 만든 `Ambience` 선택
- **"답장/전달에 사용할 서명"** → 방금 만든 `Ambience` 선택

### 3-4. 저장

페이지 맨 아래로 스크롤 → **"변경사항 저장"** 클릭

---

## ✅ 완료!

새 이메일을 작성해보면 서명이 자동으로 들어가 있습니다. 🎉
